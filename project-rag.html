Overview

End-to-end RAG system that transforms documents into intelligent, queryable knowledge bases. Features robust ingestion, vector storage, and context-aware answer generation.

  • Complete document ingestion pipeline
  • Semantic search with vector embeddings
  • RESTful API for seamless integration

Architecture

  • Ingestion: PDFs/Docs → cleaned text → smart chunks
  • Embeddings: OpenAI/Ollama → stored in pgvector
  • Retrieval: Similarity search + optional reranker
  • API: Query, add docs, health checks
  • Deploy: Dockerized services with env config
