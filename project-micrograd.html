Micrograd Implementation

A tiny deep learning framework built from first principles—scalar autograd, backpropagation, and neural networks demystified.

Python NumPy Autograd Backpropagation Neural Networks

Overview

A minimalist deep learning engine that implements scalar-valued automatic differentiation with a dynamically built DAG. Understand exactly how PyTorch and TensorFlow work under the hood.

  • Crystal-clear implementation of the chain rule
  • Educational codebase with extensive comments
  • Foundation for understanding modern DL frameworks

Key Components

  • Value class with data, grad, and backward()
  • Computational graph built dynamically via operators
  • Topological sort for correct gradient propagation
  • MLP with activations and gradient descent

Example Usage

Python 
# Build a simple neural network and train it
from micrograd import Value, MLP

# Create a multi-layer perceptron
model = MLP(n_inputs=2, sizes=[4, 1])

# Forward pass & compute loss
loss = model.loss(X, y)

# Backward pass - compute gradients
loss.backward()

# Gradient descent update
for p in model.parameters():
    p.data -= 0.01 * p.grad
