Overview

This project demonstrates parameter-efficient fine-tuning (PEFT) of open-source Large Language Models using cutting-edge techniques like LoRA and QLoRA adapters.

  • Complete training pipeline from data processing to model export
  • Task-specific adapter heads for domain adaptation
  • Comprehensive evaluation with multiple metrics

Key Features

  • QLoRA for training on limited GPU memory (4-bit quantization)
  • Domain-adapted datasets with balanced sampling and cleaning
  • Metrics: BLEU, ROUGE, BERTScore + human evaluation
  • Export merged or adapter-only weights for deployment
